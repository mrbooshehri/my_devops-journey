# Installation Vector

## RPM-based

1. Add repo
```bash
curl -1sLf 'https://repositories.timber.io/public/vector/cfg/setup/bash.rpm.sh' \
  | sudo -E bash
```

2. Install
```bash
sudo yum install vector
```

## Debain-based

1. Add repo
```bash
curl -1sLf \
  'https://repositories.timber.io/public/vector/cfg/setup/bash.deb.sh' \
| sudo -E bash
```

2. Install
```bash
sudo apt install vector
```
